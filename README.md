# 🌍 Global Restaurant Analysis Dashboard (Power BI)

## 📊 Project Overview
This project is an end-to-end Power BI dashboard built using the Zomato dataset to analyze global restaurant performance. The objective was to create a consolidated and interactive report that helps business stakeholders quickly assess performance across regions, cuisines, pricing, and customer ratings.

## 🎯 Objectives
• Analyze total restaurants globally across continents, countries, and cities  
• Identify highest-rated and lowest-cost restaurants  
• Evaluate cuisine popularity trends  
• Analyze service features like online delivery and table booking  
• Build an interactive dashboard for business decision-making  

---

## 🛠 Tools & Technologies
• Power BI (Data Modeling, DAX, Visualization)  
• Excel (Data Cleaning)  
• DAX (Measures & Calculated Columns)

---

## 📂 Dataset
The dataset contains restaurant-level information including:
• Restaurant name & location  
• Cuisines served  
• Cost for two  
• Aggregate rating  
• Online delivery & booking services  
• Geographic data (Lat/Long)

---

## ⚙️ Data Modeling
A star schema model was implemented:

• Fact Table → Zomato_Global  
• Dimension Tables → Country Master, Dim_Cuisine  
• KPI Table → Measures for reporting  

---

## 📊 Dashboard Features
✔ KPI Summary (Restaurants, Cost, Rating, Cuisine Count)  
✔ Global Restaurant Distribution Map  
✔ Top Rated Restaurants Analysis  
✔ Lowest Cost Restaurants Analysis  
✔ Cuisine Popularity Trends  
✔ Online Delivery & Table Booking Insights  
✔ Interactive Filters (Continent, Country, City, Rating Color)

---

## 🚀 Key Insights
• Asia shows the highest restaurant concentration globally  
• Majority of restaurants offer online delivery  
• Strong correlation between pricing and ratings in premium segments
